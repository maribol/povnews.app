# Chrome Web Store listing copy (draft)

## Name

POV News

## Short description

POV-scored news on your new tab — powered by your Cursor API key, no backend.

## Detailed description

POV News is a calm, reading-first new tab that scores every item against **your** pillars — not generic feeds.

- **Bring your own Cursor key** — research runs on Cursor Cloud Agents; you pay Cursor directly
- **Custom POV wizard** — pillars, sources, calibration in ~3 minutes
- **Feedback loop** — thumbs up/down trains the next digest
- **Reader mode** — extract articles in-extension with Readability
- **Export** — one-click Markdown for Obsidian or Notion

All data stays local except calls to `api.cursor.com` and article URLs you choose to open.
